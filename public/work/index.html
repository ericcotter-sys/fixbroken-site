FixBroken OS, the toolchain that enforces it, and the multi-tenant platform running underneath. The system this page is built on.

Single-file CSS design system with machine-readable JSON manifest. 15 Node.js CLI tools - voice linter, 6-category page audit, WCAG contrast checker, tenant scaffolder, manifest generator. 4 Express tenants on one Lightsail box behind nginx, systemd, webhook auto-deploy.

The system

One CSS file serves every tenant. 14 token categories, 29 components, 11 organized sections. Overrides happen per-tenant via scoped wrapper classes - never by forking. Machine-readable manifest drives the toolchain.

The toolchain

Voice linter catches banned phrases and CSS violations. Page audit scores 6 categories. Tenant scaffolder generates nginx config, systemd unit, deploy script, and override CSS in one command. Pre-commit hook blocks violations.

Platform

AWS Lightsail, Ubuntu 24.04. Each tenant is an Express server on its own port, managed by systemd, proxied by nginx. Push to main, live in 10 seconds. Real-time token telemetry at /tokenz.
